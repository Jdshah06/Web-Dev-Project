API Documentation:

The data provided by this site depicts the percentage of people voting Democrat versus the percentage of people voting Republican. Raw data can be downloaded here.

Accessing the Data:

Base Url: "https://election2020.com/api/download?
Routes: "/{STATE_NAME}"

Example:

API call: https://election2020.com/api/download?/california
